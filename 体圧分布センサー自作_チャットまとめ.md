## 🧱 材料選び

### ✅ 圧力センサー素材

-   **Velostat / Linqstat**
    -   導電性ポリマーシート
    -   圧力によって抵抗値が変化
    -   DIY体圧マットでよく使われる
-   **フォースセンシティブレジスタ（FSR）**
    -   小型で扱いやすい
    -   多数配置で圧力分布を測定可能
    -   精度は比較的低め
-   **導電性布・導電スポンジ**
    -   柔軟性が高い
    -   抵抗変化から圧力推定が可能

------------------------------------------------------------------------

## 🧩 マット構造

例: - 上部導電層 - 圧力で抵抗変化する層（Velostat等） - 下部導電層

格子状にセルを作ることで、圧力マッピングが可能。

------------------------------------------------------------------------

## 🔌 配線・読み取り方法

-   Arduino / ESP32 / Raspberry Pi Pico などのマイコンを使用
-   分圧回路で抵抗変化を電圧へ変換
-   多チャンネル読み取り:
    -   マルチプレクサ（MUX）
    -   I2C ADC（12〜16bit）

------------------------------------------------------------------------

## 📊 可視化・解析

-   抵抗 → 電圧 → 圧力への変換
-   Python + Matplotlib / OpenCV でヒートマップ表示
-   校正（重りなど）を行うと精度向上

------------------------------------------------------------------------

## 💰 コスト目安

-   Velostatシート: 約1,000円
-   FSR (16〜32個): 約1,000〜3,000円
-   マイコン: 約500〜2,000円

👉 小型体圧マットなら **5,000円以下** でも制作可能。

------------------------------------------------------------------------

## 💡 ポイント

-   高精度化するとコスト増大
-   概略の体圧分布なら低コストでも可能
-   解像度はセンサー数に依存
-   スポンジ＋Velostat構成は比較的簡単
