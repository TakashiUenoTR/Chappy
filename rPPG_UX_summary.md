# rPPG・UX評価・マルチモーダル感情推定まとめ

作成日時: 2026-02-18 09:36:33

------------------------------------------------------------------------

# 1. rPPGとは

rPPG（remote
photoplethysmography）は、カメラ映像から皮膚の色変化を解析して
心拍などの生体情報を推定する非接触技術。

主な特徴:

-   非接触
-   Webカメラで実装可能
-   リアルタイム処理が可能
-   UX研究・感情推定・HCI分野で注目

------------------------------------------------------------------------

# 2. rPPGで取得可能なデータ

## 2.1 高信頼（実用レベル）

-   心拍数（HR）
-   心拍波形
-   呼吸数（推定）
-   HRV（条件が良ければ）

## 2.2 中程度（研究〜準実用）

-   血流強度
-   ストレス指標
-   血圧推定（間接）

## 2.3 研究段階

-   SpO2推定
-   感情推定
-   血管健康指標

------------------------------------------------------------------------

# 3. rPPGでリアルタイムUX評価はどこまで可能か

リアルタイムで取得・推定可能なUX指標:

-   覚醒度（Arousal）
-   認知負荷（Cognitive Load）
-   ストレス反応
-   没入・興味（Engagement）

難しいもの:

-   感情の種類の完全識別（例：喜び vs 怒り）

------------------------------------------------------------------------

# 4. 心拍＋表情＋姿勢で精度が向上する理由

感情は以下の3軸で表現される:

1.  生理反応（心拍・HRVなど）
2.  表情（Valenceの手がかり）
3.  身体動作（姿勢・視線など）

理由:

-   心拍 → 覚醒度（強さ）に強い
-   表情 → 快/不快の方向に強い
-   姿勢 → 行動意図のヒント

モダリティ融合により精度が向上する。

------------------------------------------------------------------------

# 5. rPPG単体でも感情推定精度を上げるテクニック

-   BPMではなく波形特徴量を使用
-   HRVの時間変化を見る
-   微小変動解析（entropyなど）
-   ROI分割（頬・額など別解析）
-   空間的rPPG（血流分布）

# 8. マルチモーダル感情推定の研究的根拠

研究では以下の傾向:

-   心拍のみ: 約55〜70%
-   表情のみ: 約60〜75%
-   心拍＋表情: 約75〜90%
-   生体＋顔＋他: 約85〜95%（条件良）

参考研究例:

-   rPPG + facial expression fusion（arousal約90%）
-   EEG + 顔表情融合（約81〜82%）
-   マルチ生体信号融合（80〜95%帯）

注意:

-   データセット依存
-   制御環境の結果が多い
-   実環境では65〜85%程度が現実的

-   Python + Webカメラ1台で研究レベルのUX分析システム構築可能
