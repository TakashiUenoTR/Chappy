# UEQ + 生体計測（HRV・心拍）によるUX評価まとめ

------------------------------------------------------------------------

# 1. 概要

UX（User
Experience）は単一指標では評価できないため、以下の3層を組み合わせて評価するのが現在の主流。

## UX評価の3層モデル

① 主観評価（Self-report）\
② 行動データ（Behavioral）\
③ 生体反応（Physiological）

### ① 主観評価

-   UEQ
-   SUS
-   NPS
-   CSAT

### ② 行動指標

-   タスク成功率
-   完了時間
-   エラー率
-   操作ログ

### ③ 生体計測

-   HR（心拍数）
-   HRV（心拍変動）
-   EDA（皮膚電気反応）
-   視線

------------------------------------------------------------------------

# 2. UEQ（User Experience Questionnaire）

## 概要

UXを多次元評価できるアンケート。

-   無料
-   研究・実務で広く使用
-   UXの感性要素を数値化可能

## UEQの6評価軸

  スケール         意味
  ---------------- ----------------------
  Attractiveness   総合的な魅力
  Perspicuity      分かりやすさ
  Efficiency       効率性
  Dependability    信頼性・操作の安心感
  Stimulation      楽しさ・刺激
  Novelty          新規性

## スコア形式

-   7段階評価（対義語ペア）
-   スコア変換： -3 ～ +3

## スコア目安

  スコア   評価
  -------- ------------
  +2以上   非常に良い
  +1～2    良い
  -1～+1   普通
  -1以下   問題あり

------------------------------------------------------------------------

# 3. なぜUEQ + 生体計測が強いのか

## UX評価の問題

アンケートのみ：

-   記憶バイアス
-   社会的バイアス
-   無意識感情を測れない

生体のみ：

-   意味解釈が難しい

## 解決策

主観（UEQ） + 客観（生体） = 高精度UX評価

------------------------------------------------------------------------

# 4. UEQと相性の良い生体指標

## HRV（心拍変動）

### RMSSD / HF

意味：

-   リラックス
-   快適性

関連UEQ：

-   Attractiveness
-   Stimulation

### LF/HF

意味：

-   ストレス
-   認知負荷

関連UEQ：

-   Efficiency
-   Perspicuity

## 心拍数（HR）

意味：

-   興奮
-   緊張
-   集中

### 解釈例

  状態                        解釈
  --------------------------- ------------
  HR↑ + Stimulation↑          良い興奮
  HR↑ + Efficiency↓           UXストレス
  HRV安定 + Attractiveness↑   心地よいUX
